| GitHub Actions runner → GitHub Pages CDN | Workflow has `pages: write` + `id-token: write` — broad enough to publish but no broader. |
| Live URL `https://elsolarcg.github.io/planetapsilo/` → public visitors | `noindex,nofollow` + `robots.txt Disallow: /` keep search engines out, but humans with the URL can reach the site. Acceptable per CONTEXT.md ("sitio existe pero NO se indexa hasta Phase 3"). |
| Workflow permissions block → GitHub API | `contents: read` + `pages: write` + `id-token: write` is the documented minimum scope per actions/deploy-pages docs. |
